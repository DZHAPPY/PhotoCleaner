# 相册清理助手

一个面向 Android 的相册整理应用。  
它会读取系统相册中的图片，帮助用户用更轻松的方式完成清理:

- 随机抽取照片，一张张决定“保留 / 删除 / 跳过”
- 自动识别截图、相似照片、模糊照片，支持集中清理
- 统计照片数量、占用空间、累计删除数量
- 支持删除确认、整理记录、日间 / 夜间界面切换

这个项目的目标，不只是“清理相册”，也是让整理照片这件事变得更有回忆感、更不容易拖延。

## 功能简介

### 1. 随机整理

应用会从系统相册中随机抽取一批照片，逐张展示在界面中。  
用户可以对每张照片执行以下操作：

- 保留
- 删除
- 跳过
- 撤销上一步

这种方式适合在碎片时间里慢慢整理，也能随机翻到过去的照片。

### 2. 智能清理

首页提供建议清理入口，目前支持：

- 截图清理
- 相似照片建议去重
- 模糊照片建议删除

用户可以批量选择并统一删除。

### 3. 相册统计

首页会显示：

- 照片总数
- 占用空间
- 时间跨度
- 累计已删除照片数量

### 4. 删除流程

应用使用 Android 系统媒体库接口删除图片，删除时会弹出系统确认框。  
删除完成后会自动重新扫描相册，刷新统计信息和后续整理内容。

### 5. 整理记录

应用会记录整理历史，包括：

- 每次整理的时间
- 删除数量 / 释放空间
- 最近整理记录
- 连续活跃天数与徽章进度

### 6. 主题切换

支持日间 / 夜间两套界面风格，适配不同使用环境。

## 适用场景

- 相册照片很多，不知道从哪里开始删
- 想先把截图、模糊图、重复感强的照片清掉
- 希望整理相册时顺便回顾旧照片
- 想用更轻量、更碎片化的方式完成相册断舍离

## 技术实现

项目基于原生 Android 开发，当前主要使用：

- Kotlin
- Android SDK
- Material Components
- ConstraintLayout
- MediaStore

核心实现思路包括：

- 使用 `MediaStore` 读取系统图片资源
- 使用系统删除请求删除照片
- 本地统计整理记录与累计数据
- 基于启发式规则做截图、相似图、模糊图建议识别

## 开发环境

- Android Studio
- JDK 11
- Android `minSdk 33`
- `targetSdk 36`

## 运行方式

### 方式一：Android Studio

1. 使用 Android Studio 打开项目目录
2. 等待 Gradle 同步完成
3. 连接 Android 设备或启动模拟器
4. 运行应用

### 方式二：命令行构建

```bash
./gradlew assembleDebug
```

调试包输出路径：

```text
app/build/outputs/apk/debug/app-debug.apk
```

## 权限说明

当前应用需要图片读取权限：

- `READ_MEDIA_IMAGES`

该权限仅用于读取系统相册中的图片并完成整理、统计、删除操作。
